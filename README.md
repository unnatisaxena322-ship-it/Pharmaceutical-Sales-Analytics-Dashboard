# Pharmaceutical-Sales-Analytics-Dashboard
Interactive Power BI dashboard for pharmaceutical sales performance analysis.
Pharmaceutical Sales Analytics Dashboard

Overview

Interactive Power BI dashboard developed to analyze pharmaceutical sales performance across products, brands, regions, and sales channels.

Tools Used

- Power BI
- Microsoft Excel

Dashboard Features

- Revenue KPI Tracking
- Gross Profit Analysis
- Monthly Revenue Trend Analysis
- Revenue Distribution by Sales Channel
- Top Brands by Revenue
- Interactive Filtering with Slicers

Key Skills Demonstrated

- Data Visualization
- Dashboard Development
- Data Modeling
- Business Analytics
- Power BI Reporting
